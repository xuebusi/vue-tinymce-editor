# vue-tinymce-editor

TinyMCE富文本编辑器在Vue3中的使用

## 配置步骤
**1.将tinymce静态资源放到public目录中**
**2.在components下创建Editor组件**
**3.在main.js中注册Editor组件**
**4.载创建文章对话框中使用Editor组件**
**5.在App.vue中使用文章组件**

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
